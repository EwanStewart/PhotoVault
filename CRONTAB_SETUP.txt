Photoframe Display Schedule Setup
==================================

To enable automatic display on/off scheduling:

1. SSH to the Raspberry Pi

2. Make the script executable:
   chmod +x /home/ewastewa/photoframe/display-schedule.sh

3. Add to crontab:
   crontab -e

4. Add this line at the end:
   * * * * * /home/ewastewa/photoframe/display-schedule.sh

5. Save and exit

The display will now:
- Turn OFF at 9:00 PM (21:00)
- Turn ON at 7:00 AM (07:00)

Note: The script checks every minute but only changes state when crossing
the time boundaries. Manual display toggling (via double-tap) still works
and will be overridden at the next schedule boundary.
